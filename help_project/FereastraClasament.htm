Fereastra Clasament, dupa cum ii spune si numele, afiseaza clasamentul fiecarei categorii de concurs. Aceasta contine aceleasi informatii ca si fereastra Concurenti, doar ca biciclistii sunt afisati in ordinea pozitiei din clasament.

O alta diferenta intre aceasta fereastra si fereastra Concurenti este ca in aceasta fereastra penalizarile nu pot fi modificate, ele fiind doar afisate. Introducerea acestora se face din fereastra Concurenti.
